# FUTURE_CS_02

# Phishing Detection & Awareness

## Introduction

For this assessment, I analyzed a collection of phishing, suspicious, and legitimate email samples to identify common phishing indicators and evaluate the level of risk associated with each message. The objective of this assessment was to strengthen my ability to recognize phishing attempts, communicate security risks in a business-friendly manner, and recommend practical measures that help users avoid phishing attacks.
